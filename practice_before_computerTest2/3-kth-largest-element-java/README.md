#Kth Largest Element（20 分）

## 题目:
findKthLargest方法输入一个非空整数数组 和 整数 k，返回数组中第k大的数。假设k是有效的。1<=k<=数组长度。


## 公开测试用例：(公开测试用例也是计分的! 另外有 15 个非公开测试用例)
参数:[1, 2, 3, 4, 5, 6],2
返回:5

参数:[3, 2, 1, 5, 6, 4],3
返回:4

参数:[3, 2, 1, 5, 6, 4],6
返回:1

参数:[1, 1, 2, 2, 3, 3],2
返回:3

参数:[6, 5, 4, 3, 2, 1],1
返回:6
